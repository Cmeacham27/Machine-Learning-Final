# Machine-Learning-Final
This project analyzes U.S. cancer death rates by state from 2014–2023 to find trends, patterns, and build models to try and predict future results. The dataset includes yearly cancer mortality rates, which were used to see how rates change over time and how accurately future rates can be predicted.

## Problem Statement & Goals
The main goal of this project was to determine:
Whether cancer death rates follow a predictable trend over time. How accurately we can predict future cancer death rates using historical data. Whether meaningful patterns exist across states and regions

## Overall Approach
The project followed a structured data science pipeline:
Data collection and cleaning
Exploratory Data Analysis
Feature scaling and preprocessing
Predictive modeling (regression + classification)
Clustering and dimensionality reduction (K-Means + PCA)
Model evaluation and interpretation

Multiple models were tested, including:
Linear Regression
Decision Tree Regressor
Support Vector Machines (SVR & SVC)
Random Forest
Neural Networks (MLP)

## Project Sections
RAW_DATA: Data source and acquisition
DATA: Cleaning, preprocessing, and visualization
ANALYSIS: Modeling and technical evaluation
CONCLUSIONS: Key findings and insights

## Summary
Through this project, I learned that cancer death rates in the U.S. are very stable and follow a consistent downward trend. This makes them very helpful for making predictive modeling. I also learned that while total deaths may increase, death rates can still decline, which provides a more accurate measure of health outcomes. This is because population can grow throughout the year, but the rate can stay the same, or go down. Additionally, different models vary in effectiveness, with neural networks creating more complex patterns than simpler models.
